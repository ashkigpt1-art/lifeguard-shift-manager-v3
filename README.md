# مدیریت شیفت ناجیان

سامانه‌ی تمام‌عیار برای برنامه‌ریزی شیفت ناجیان با FastAPI و React (Vite + TypeScript).

## اجرای سریع

```bash
docker compose up --build
```

## بک‌اند

```bash
cd backend
uvicorn app.main:app --reload --port 8000
```

## فرانت‌اند

```bash
cd frontend
npm install
npm run dev
```

تست‌ها:

```bash
cd backend
pytest
```
